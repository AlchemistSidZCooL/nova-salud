# NOVA Health: Nexus Protocol

Este sistema es el núcleo de gestión de salud del ecosistema **NOVA**, diseñado para sincronizar la vida clínica de Daniel Hernán García Alzate con su entorno neurodivergente y profesional.

## Estado del Sistema
- **Versión:** 4.3.1 "Nexus Evidence (Rectified)"
- **Último Hito:** Procesamiento del zip `Photos-3-002.zip` con evidencias de la cita dental del 24/04.
- **Próximo Hito:** Protocolo post-operatorio y seguimiento de la medicación (Amoxicilina/Enantyum).
- **Próximo Paso Crítico:** Evaluación Psiquiátrica (TDAH/TEA) mediante el sistema **N-Log**.

## Características Principales
- **Protocolo N-Log:** Sistema estratégico de etiquetado (`[EXE]`, `[SEN]`, `[SOC]`, `[FOC]`) para la recolección de evidencia clínica de síntomas TDAH/TEA.
- **Evidencias Digitalizadas:** Galería indexada de capturas clínicas reales (Recetas, Presupuestos, Pautas post-op) para trazabilidad total.
- **Sincronización Dual de Relojes:** Monitorización en tiempo real de Madrid y Colombia, incluyendo estados de actividad social remota.
- **Traductor de Fases Médicas:** Simplificación de términos técnicos para una toma de decisiones informada.

## Portabilidad y Migración
El sistema cuenta con un archivo de esquema **[NOVA_Health_Schema.json](file:///home/sidzcool/GeminiSolutions/NOVA_proyect/salud/NOVA_Health_Schema.json)** que permite:
1.  **Migración:** Exportar la lógica de los protocolos a otras herramientas de gestión (Notion, Trello, Jira).
2.  **Interoperabilidad:** Ser interpretado por otros agentes de IA o sistemas automatizados.
